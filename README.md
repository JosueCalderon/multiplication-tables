# NOTES:

Dynamic multiplication tables with Node.js

```
Options: 
    --help     Show help                                            [booleano]
    --version  Show version number                                  [booleano]
-b, --base     Multiply base                              [número] [requerido]
-l, --list     Show table in console                                [booleano]
-e, --end      Maximum number of multiplications          [número] [requerido]

```
